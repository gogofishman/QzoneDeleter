## 预览

![](https://p.sda1.dev/13/e2bd534ba8b8f16ad03110f40adac3d9/预览图.jpg)
## 功能

* 一件批量删除空间说说（700毫秒左右删除一条）
* 移除flash悬浮窗
* 移除开通黄钻
* 移除签到
* 移除天气
* 移除小游戏
* 移除无用信息栏
* 优化左侧tab栏

## 使用
1. 点击页面右上角的“del”按钮打开或隐藏面板
2. 在空间（刚点进去的界面）或者说说界面，点击启动即可

## 注意

短时间内删除多条说说（几百到一千多左右），空间会启动防御机制，要求你删除时需要验证，此时会自动停止，你最好等几个小时或几天再操作。
